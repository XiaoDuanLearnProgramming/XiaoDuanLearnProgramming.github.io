java知识清单-多态

用法

- 父类(或接口) 对象名 = new 子类()

多态的体现

- 父类的引用指向自己子类的对象
- 父类的引用也可以接收自己的子类的对象

多态的前提

-必须是类与类之间有继承关系或者是类与接口之间的实现关系
-子类要对父类中的方法存在覆盖，不然该多态的实现是没有意义的

多态的好处

- 提高代码的扩展性
- 提高代码的复用性

多态的应用

多态的局限性

- 只能使用父类中的引用访问父类中的成员

注意事项：

- 实现多态时是对象的类型提升，向上转型故只能调用父类（或接口）中已有的方法；如果想要调用子类中特有的方法，需强制将父类的引用，转成子类类型，即向下转型。
- 不要出现将父类对象转换为子类类型
